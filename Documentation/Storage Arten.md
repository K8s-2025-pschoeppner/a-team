# Storage Arten
### Ephemeral Storage

Ephemeral Storage (vergänglicher Speicher) bleibt nicht über die Lebensdauer eines Containers hinaus bestehen. Wenn ein Container neu gestartet wird, gehen alle Daten im Ephemeral Storage verloren.
